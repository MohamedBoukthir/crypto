# JavaFX Cryptography

## Prerequisites
- **JDK 21** or later
- **Maven 3.6.0** or later

## Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MohamedBoukthir/crypto.git
   cd crypto
   ```

2. **Build the project:**
   ```bash
   ./mvnw clean install
   ```

3. **Run the application:**
   ```bash
   ./mvnw javafx:run
   ```
